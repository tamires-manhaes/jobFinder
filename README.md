# JobFinder

 > - ***requisitos:***
   > NodeJs : 14.2.0 ou mais atual
   > Yarn: 1.22.4 >ou mais atual
>  - **Para instalar app:**
>   -> *yarn install*
> - **Para rodar:**
>   -> *yarn dev*

> - ***rotas de admin:***
> **/jobs/admin** : vizualiza todas as vagas em formato JSON
> **/jobs/remove** : view para deletar vaga pelo id
